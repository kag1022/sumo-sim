# 力士育成シミュレーション（仮）仕様書

## 1. 概要 (Overview)
プレイヤーは相撲部屋の親方となり、**たった一人の弟子**を手塩にかけて育て上げ、最高位「横綱」を目指す育成シミュレーションゲームです。
従来的な「多人数管理型」とは異なり、一人の力士の人生（入門から引退、そして継承まで）に深く寄り添うゲームデザインを特徴とします。

---

## 2. コアゲームループ (Core Game Loop)

### A. 部屋経営・育成フェーズ (Stable Mode)
本場所開催までの準備期間です。「残り週数（ターン）」を消費して、力士に直接指導を行います。

*   **直接指導 (Direct Instruction)**:
    *   親方として、コマンドを選択して弟子を指導します。
    *   **身体強化**: 筋力・体重を強化。疲労大。
    *   **技術指導**: 技術・瞬発力を強化。疲労中。
    *   **対話**: 信頼度・やる気を向上。
    *   **休養**: 疲労回復・コンディション調整。
*   **パラメータ管理**:
    *   **やる気 (Motivation)**: トレーニング成功率や試合のパフォーマンスに影響。
    *   **信頼度 (Trust)**: 親方への信頼。低いと指示を聞かないことがある。
    *   **調子 (Condition)**: 絶好調〜不調の4段階。試合勝率に大きく影響。
*   **ターン制**:
    *   コマンドを実行すると、本場所までの残り週数が減少します。
    *   残り週数が0になると、本場所へ出場します。

### B. 本場所フェーズ (Tournament Mode)
15日間の本場所を開催します。

*   **取組**:
    *   幕下以下は7番、十両以上は15番の取組を行います。
    *   オート進行、または手動で日程を進められます。
*   **勝敗決着**:
    *   力士の能力値、当日の調子、やる気、そして「運」が絡み合う試合エンジン。
    *   決まり手や対戦相手との相性（ライバル関係）も考慮されます。
*   **番付編成**:
    *   場所終了後、勝ち越し・負け越し数に基づき、厳密な昇降格判定を実施。
    *   **横綱・大関昇進**: 厳しい基準（大関33勝、横綱2連続優勝準拠）をクリアする必要があります。

## 3. ライフサイクル (Lifecycle)

本作の最大の特徴は、力士の「一生」を描くことです。

### A. 入門 (New Game / Succession)
*   **新弟子登録**:
    *   ゲーム開始時、または引退時に、新しい弟子の「四股名」と「年齢（15〜25歳）」を自由に設定できます。
    *   新弟子は**序ノ口 (Jonokuchi)** からキャリアをスタートします。

### B. 加齢と衰え (Aging & Decline)
*   **加齢**: 6場所（約1年）ごとに年齢が1つ上がります。
*   **ピーク年齢 (Peak Age)**:
    *   力士ごとに成長のピーク（25〜29歳前後）が設定されています。
    *   ピークを過ぎると、トレーニング効果が減少し、徐々に能力が衰え始めます。

### C. 引退 (Retirement)
*   **引退条件**:
    1.  **手動引退**: プレイヤーの判断でいつでも引退を勧告できます。
    2.  **能力限界**: ピークを過ぎ、番付を維持できなくなった場合（例：大関陥落、十両から幕下転落）。
    3.  **横綱の責任**: 横綱昇進後、キャリア通算で2回「負け越し」を喫すると、責任を取って即引退となります。
*   **殿堂入り (Hall of Fame)**:
    *   引退した力士は、その実績に応じて「Legend」「Star」「Pro」「Local Hero」の称号と共に記録されます。
*   **継承**:
    *   引退後、直ちに「新弟子登録画面」へ移行し、次の世代の力士育成が始まります。

## 4. データモデル (Data Model)

### 力士 (Rikishi)
*   **基本情報**: 四股名、年齢、ランク（詳細表記：東序ノ口1枚目など）。
*   **能力値**: Strength, Weight, Speed, Technique, Mental (0-200スケール)。
*   **状態値**:
    *   Motivation (0-100)
    *   Trust (0-100)
    *   Condition (Enum: ZEKKOCHO...BAD)
    *   Fatigue (0-100)
*   **特性 (Traits)**:
    *   「土俵際」「ロケットスタート」「晩成」など、個性を彩るパッシブスキル。

### 世界観 (World)
*   **45の相撲部屋**: プレイヤー以外の45部屋がAIとして存在。
*   **黄金時代**: ランダムなAI部屋が「黄金時代」を迎え、強力なライバルを輩出します。
*   **ライバル**: 何度も負けた相手は「宿敵」として記録され、対戦時の緊張感が高まります。

## 5. UIについて
*   **Instruction View (ホーム画面)**:
    *   力士の立ち絵、詳細パラメータ、指導コマンドを集約。
    *   常に自分の弟子の状態を一目で把握できるデザイン。
*   **Setup View**:
    *   シンプルな新弟子登録インターフェース。
