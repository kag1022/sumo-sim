# Sumo Simulation Game (大相撲部屋経営シミュレーション)

## 概要
プレイヤーは相撲部屋の親方となり、力士をスカウトし、育成し、最高位「横綱」を目指す経営シミュレーションゲームです。
リアルな番付システム、昇進・陥落のドラマ、そして部屋の経済状況を管理しながら、最強の部屋を作り上げてください。

UIデザインは **"Sumo Modern Traditional"**（相撲モダン・トラディショナル）をテーマに、和紙の質感や伝統的な色彩と、現代的な管理画面の利便性を融合させています。

## 主な機能

### 1. ゲームモード選択
ゲーム開始時に以下から選択可能です。
- **独立 (Establish)**: 資金300万、弟子1名からスタートする高難易度モード。
- **継承 (Inherit)**: 資金1500万、弟子6名を引き継いでスタートする標準モード。

### 2. スカウトと新弟子検査
将来の有望株を見つけるための重要なフェーズです。
- **スカウト**: 日本全国・海外から候補者がリストアップされます。出身地や格闘技歴、体格(身長・体重)を確認できます。
- **新弟子検査 (有料)**: 
  - 費用: **¥300,000**
  - 検査を行うことで、隠された**素質 (S〜C評価)** と **柔軟性** が判明します。
  - 素質が高いほど、トレーニングによる能力の伸びが良くなります。
- **命名 (四股名)**: 入門時に、プレイヤーが自由に四股名を付けることができます。
- **前相撲**: 
  - 新弟子はまず「前相撲」として入門します。翌場所、自動的に「序ノ口」に昇進し、公式戦デビューを果たします。
- **自動獲得機能**: 他部屋への入門を自動化する設定が可能。

### 3. 育成システムと秘技
本場所のない期間（偶数月など）は、週単位で稽古メニューを指示します。
- **四股 (基礎固め)**: 「体」ステータス重視。怪我リスク極低。
- **鉄砲 (技術研鑽)**: 「技」ステータス重視。怪我リスク極低。
- **申し合い (実戦形式)**: 全ステータスが大きく伸びるが、怪我のリスクがある。
- **特別指導と秘技**: 
  - 親方の指導力（トレーニングポイント）を使い、個別に「特別指導」を行えます。
  - 稀に強力な**「秘技（スキル）」**（鉄の額、巨漢殺し、うっちゃり等）をひらめくことがあります。
  - 秘技は取組中に発動し、勝率に影響を与えたり、決まり手を変化させます。

### 4. 本場所 (トーナメント)
奇数月は本場所が開催され、15日間の熱戦が繰り広げられます。
- **取組**: スイス式トーナメント準拠のマッチング。
- **勝敗**: 「心・技・体」＋「秘技」＋運要素で決まります。
- **全階級優勝決定**: 幕内から序ノ口まで、各段で優勝者が決定されます。
- **賞金**: 幕内最高優勝を果たすと、賞金 **1,000万円** が部屋に入ります。
- **記録室 (History)**: 歴代の優勝者や、西暦（2025年〜）を含めた詳細な記録を閲覧できます。

### 5. 番付と進退、引退ドラマ
- **6つの階級と定員**: 幕内・十両・幕下・三段目・序二段・序ノ口。
- **関取**: 十両以上の力士。給金が発生し毎日取組があります。
- **引退システム**: 
  - **引退相談**: 力士が引退を考え始めた時、「荘厳な夜」をテーマにした相談イベントが発生。引き止めることも可能です。
  - **断髪式**: 引退時には「大儀式」としての断髪式を執り行います。鋏を入れ、新たな門出「祝」を送り出します。
  - **功労金**: 最高位と在籍期間に応じた功労金が得られます。

### 6. 部屋経営 (経済)
- **経営手腕**: 
  - **女将さん**: レベルアップすることで、力士のストレスケア効率が向上します。
  - **施設**: 稽古場の設備を強化し、育成効率を上げることができます。
  - **大福帳UI**: 経営状態は、「商家の大福帳」を模したモダンな管理画面で確認できます。

## デザインテーマ "Sumo Modern Traditional"
- **Color**: ベンガラ色（#b7282e）を基調とし、和紙色（#fcf9f2）を背景に使用。
- **Typography**: 明朝体（Serif）をタイトルに使用し、伝統的な雰囲気を演出。
- **Metaphors**: 「看板」「大福帳」「儀式」などの伝統的メタファーをUI各所に採用。

## 技術スタック
- **Frontend**: React (Vite), TypeScript
- **Styling**: TailwindCSS
- **State Management**: React Context API
- **Build**: Vite

## インストールと実行
```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
