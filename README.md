# Sumo Simulation Game (大相撲部屋経営シミュレーション)

## 概要
プレイヤーは相撲部屋の親方となり、力士をスカウトし、育成し、最高位「横綱」を目指す経営シミュレーションゲームです。
リアルな番付システム、昇進・陥落のドラマ、そして部屋の経済状況を管理しながら、最強の部屋を作り上げてください。

## 主な機能

### 1. スカウトと新弟子検査
将来の有望株を見つけるための重要なフェーズです。
- **スカウト**: 日本全国・海外から候補者がリストアップされます。出身地や格闘技歴、体格(身長・体重)を確認できます。
- **新弟子検査 (有料)**: 
  - 費用: **¥300,000**
  - 検査を行うことで、隠された**素質 (S〜C評価)** と **柔軟性** が判明します。
  - 素質が高いほど、トレーニングによる能力の伸びが良くなります。
- **命名 (四股名)**: 入門時に、プレイヤーが自由に四股名を付けることができます。
- **前相撲**: 
  - 新弟子はまず「前相撲」として入門します。
  - 入門直後の場所では取組を行わず、育成に専念します。
  - 翌場所、自動的に「序ノ口」に昇進し、公式戦デビューを果たします。

### 2. 育成システム
本場所のない期間（偶数月など）は、週単位で稽古メニューを指示します。
- **四股 (基礎固め)**: 「体」ステータス重視。怪我リスク極低。
- **鉄砲 (技術研鑽)**: 「技」ステータス重視。怪我リスク極低。
- **申し合い (実戦形式)**: 全ステータスが大きく伸びるが、怪我のリスクがある(5%)。
- **休養**: 怪我の回復や疲労抜きを行います。

### 3. 本場所 (トーナメント)
奇数月は本場所が開催され、15日間の熱戦が繰り広げられます。
- **取組**: 同じくらいの成績の力士とマッチングされます（スイス式トーナメント準拠）。
- **勝敗**: 「心・技・体」の総合力と、多少の運要素で決まります。
- **全階級優勝決定**: 幕内から序ノ口まで、各段で優勝者が決定されます。
- **賞金**: 幕内最高優勝を果たすと、賞金 **1,000万円** が部屋に入ります。

### 4. 番付と進退
- **6つの階級**: 幕内・十両・幕下・三段目・序二段・序ノ口。
- **昇進・陥落**: 場所ごとの成績（勝ち越し・負け越し）により、厳格に番付が変動します。
- **関取**: 十両以上の力士は「関取」と呼ばれ、給金が発生し、毎日取組があります。幕下以下は7番のみです。
- **引退**: 
  - 年齢による衰えや、怪我、成績不振により力士は引退します。
  - プレイヤーは任意のタイミングで所属力士に引退勧告を行えます。
  - 引退時には、最高位と在籍期間に応じた「功労金（ご祝儀）」が部屋に入ります。

### 5. 部屋経営 (経済)
- **初期資金**: ¥10,000,000
- **収入**: 優勝賞金、引退時のご祝儀、など。
- **支出**: スカウト契約金、新弟子検査費用、給与など。
- 資金が枯渇しないよう、有望な力士を育てて賞金を稼ぐか、適切な新陳代謝(引退とスカウト)が必要です。

## 技術スタック
- **Frontend**: React (Vite), TypeScript
- **Styling**: TailwindCSS
- **State Management**: React Context API
- **Build**: Vite

## インストールと実行
```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
